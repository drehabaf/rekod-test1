# Senarai Pesakit Drehab AF - Next.js

Repo Next.js siap deploy ke Vercel.

## Cara run local

```bash
npm install
npm run dev
```

## Cara deploy Vercel

1. Upload semua fail ke GitHub repo.
2. Buka Vercel.
3. Import repo.
4. Framework preset: Next.js.
5. Deploy.

## Nota

Data disimpan menggunakan localStorage browser.
